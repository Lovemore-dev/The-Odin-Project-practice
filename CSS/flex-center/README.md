# Centering a div

This is a simple css flex exercise from the Odin Project course that I am using for practice.
The task just simply requires a a simple div which should be centered

## Techniques used

I created two containers

> One for the background

> One for the center div

So I used this in the container

    display: flex;
    justify-content: center;
    align-items: center;
